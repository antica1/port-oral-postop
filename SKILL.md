## 术后 PORT 的独特挑战

### 术前 vs 术后——根本差异

| | 术前（根治性 RT） | 术后（PORT） |
|---|---|---|
| **靶区来源** | 影像学可见肿瘤 + 亚临床扩展 | **手术床 + 病理报告** |
| **解剖** | 正常解剖 | **解剖变形**（皮瓣/缝合/水肿） |
| **核心文件** | MRI | **手术记录 + 病理报告** |
| **外放** | GTV + N mm | **手术床 + N mm（不同亚区不同）** |
| **关键风险** | 地理漏照 | **漏照 + 过度覆盖**（双方向） |

> **术后靶区勾画的黄金法则：手术记录和病理报告比 MRI 更重要。**

## 一、术前三件事——不看手术记录不画靶区

### 手术分三档

| 档 | 术式 | 靶区含义 |
|----|------|---------|
| 🟢 **规范** | 标准根治术+皮瓣+该清的区域全清 | 按手术床外放即可 |
| 🟡 **次规范** | 原发灶切了，但某区域没清 | ⚠️ 没清的区域是"黑洞" |
| 🔴 **不完整** | 仅原发灶切除，未做区域清扫 | 按未手术的靶区原则处理 |

### 最危险的次规范术式——舌癌"遗漏口底"

```
标准 舌颌颈联合根治：舌切除 ✓ + 口底清扫 ✓ + 颈清 ✓ → 完整
次规范 "舌颈"术式：舌切除 ✓ + 颈清 ✓ + 口底 ╳ → 口底是盲区
```

**口底遗漏 → 双重风险**：① 口底软组织镜下残留 ② 颈清阻断顺行引流→逆流至对侧颈

**口底遗漏 → 靶区补偿**：全口底纳入 CTV + 对侧口底/Ia/Ib + 外放 1.5→2 cm + 剂量不降

> **口底铁律**：没有明确写"口底清扫"或"口底切除"→口底就是盲区→必须照。

### 过站淋巴结——手术从来不清，颈清后成为逆流靶

| 过站淋巴结 | 解剖位置 | 颈清后为什么危险 |
|----------|---------|----------------|
| **VIII 区（腮腺/耳前）** | 腮腺浅叶内及周围 | Ib/II 被清→逆流至腮腺 |
| **IX 区（面动脉/颊淋巴结）** | 面动脉旁、颊肌外侧 | Ib 被清→面动脉旁淋巴结逆流受侵 |
| **X 区（枕后/耳后）** | 枕后三角 | V 区被清→逆流可能 |
| **颈鞘上段** | 颈动脉鞘至颈静脉孔/颈动脉管 | 负荷重→沿鞘逆行至颅底 |

### VIII/IX 纳入 CTV 的适应症

| 原发部位 | VIII 区 | IX 区 |
|---------|:--:|:--:|
| 后颊 | ✅ | ✅ |
| 上颌牙龈（后部） | ✅ | ✅ |
| RMT | ✅ | ✅ |
| 前颊 | — | ✅ |
| 下唇外侧 | — | ✅ |
| 上唇 | ✅ | ✅ |
| 口底/舌（前 2/3） | — | — |

> 🔴 **颊癌 + 同侧颈清 → IX 必照。手术后 IX 区从未被清扫——必须由放疗靶区覆盖。**

### 颈鞘逆流——何时覆盖至颅底

| 同侧 LN 状态 | 颈鞘至颅底 | 
|------------|:--:|
| N0-N1, 1-2枚小 LN, 无 ENE | ❌ |
| ≥3枚阳性 LN / ≥3cm / ENE(+) | ⚠️ 50-54 Gy |
| ≥3枚 + ENE(+) + 大块 | 🔴 必须覆盖 |

颈鞘勾画：下界=受累 LN 层面，上界=颈静脉孔+颈动脉管外口，宽度=鞘周5-8mm，剂量 50-54 Gy。

### 咽后淋巴结（RP）——口腔癌何时需要照

| 口腔癌部位 | RP？ | 理由 |
|----------|:--:|------|
| 舌前2/3 / 口底 / 前颊 / 硬腭前1/2 / 下牙龈前 | ❌ | 不引流至 RP |
| **硬腭后1/2 / RMT / 后颊→翼下颌缝 / 最后一颗臼齿牙龈 / 舌缘偏后** | ⚠️ 同侧 RP 外侧组 | "后门"→咽侧壁→RP |

> **口腔癌 RP 不常规照——除非到了"后门"。一旦到后门，RP 外侧组必须纳入。**

### 皮肤侵犯、气管切开、跳跃路径、肌肉回缩

**皮肤侵犯**：受侵→全层+Bolus 0.5-1cm+切缘+2cm；溃烂→外放 3cm。

**气管切开**：造口全层纳入 CTV + 周围 1-2cm 皮下。

**跳跃路径**：舌→直接跳跃至 IV；Ia→直接 III 前缘；I→皮下→VI；颊→双侧 IX。

**肌肉回缩**：翼内肌远心端先切（下颌角附着切断）→肌肉向上回缩→肿瘤种子被拉入颞下窝→**CTV 上界必须至颧弓+颞下窝**。手术记录无描述→默认按远心端先切处理。

### 典型病例：上前牙牙龈癌——手术改变淋巴引流的范例

> 这是理解"为什么口腔癌 PORT 必须覆盖 IX/VIII/对侧"的最佳教学病例。

```
患者：前上门齿（11-12 牙位）牙龈癌，累及上唇和上前颊

手术：上颌骨前部切除（因骨厚，无法根治性全切→避免鼻梁塌陷）
      仅同侧 Ib+II 选择性颈清扫
```

**三重手术不彻底性：**
① 骨切缘不彻底 → CTV 必须覆盖残余上颌骨 + 鼻底黏膜
② 唇/颊黏膜下淋巴管网 → 可**向双侧**引流至 IX 区
③ 仅单侧颈清扫 → 同侧顺行引流被阻断

**手术改变引流后的三条路径：**

```
术前（生理状态）：
  上牙龈→同侧 Ib → II → III（顺行）
  唇/颊→双侧 IX → Ib → II（顺行）

术后（颈清扫后）：
  路径 A：同侧 IX → 同侧 Ib（已被清扫阻断）→ ╳
  路径 B：同侧 IX → 同侧 Ib（阻断）→ 逆流至同侧 VIII（腮腺）
  路径 C：双侧 IX → 对侧 IX → 对侧 Ib → 对侧 II（对侧没清→通道完全开放！）
```

**靶区必须补偿的四个靶点：**

| 靶点 | 为什么 | CTV 动作 |
|------|------|---------|
| 残余上颌骨 + 鼻底 | 骨切缘不彻底 | 全层残余上颌骨前部 + 鼻底黏膜 ≥1 cm |
| **双侧 IX 区** | 唇/颊皮下淋巴网→双侧 IX | 面动脉旁、颊肌外侧 |
| **同侧 VIII 区** | 同侧颈清扫→逆流至腮腺 | 同侧腮腺浅叶淋巴结 |
| **对侧 Ib+II** | 单侧颈清→对侧通道完全开放 | 对侧 Ib+II |

> **核心教训**：如果只看"上颌骨切除+同侧颈清"按常规画靶区→漏掉双侧 IX、同侧 VIII、对侧 Ib+II。手术改变了淋巴地形图，放疗需要用靶区重新绘制这张地图。

> **间室放疗原则**：肿瘤未侵犯但手术动过的间室→必须全间室覆盖。完整间室清单见 `头颈部间室放疗_全部筋膜间隙清单.md`。

### 术前影像融合

CTV = "重建术前 GTV-P + 1 cm" 与 "手术缺损区/皮瓣 + 0.5 cm" 的**复合体积**（九院进修班教学）。在 TPS 上形变配准术前 MRI→术后 CT，GTV-P 投影显示皮瓣下真实位置。

## 二、口腔癌——按亚部位精确定义

### 2.0 亚部位完整清单

| 部位 | 亚部位 | 关键差异 |
|------|--------|---------|
| **唇** | 上唇/下唇/口角 | 上唇→VIII/Ib；下唇→Ia→Ib |
| **舌** | 舌尖/舌背/舌腹/舌缘/偏前vs偏舌根 | 舌尖→Ia→Ib；舌根→II/III |
| **口底** | 侧口底/前口底 | 侧→同侧Ib+II；前→双侧Ia+Ib |
| **牙龈** | 上/下+牙位+颊侧/舌侧 | 颊侧→颊黏膜；舌侧→口底/舌 |
| **颊黏膜** | 前颊/后颊 | 前颊→Ib+II；后颊→II+III |
| **硬腭** | 前/中/后 | 后部→可能软腭方向扩展 |
| **RMT** | 磨牙后三角 | 沿翼下颌缝向上→颅底 |

### 2.1 唇癌

| 亚部位 | CTV | 颈部 |
|--------|-----|------|
| 下唇（外侧） | 手术床+全层唇组织 | Ia+Ib |
| 下唇（近中线） | 同上 | 双侧 Ia+Ib |
| 上唇 | 手术床+VIII+Ib | VIII+Ib |
| 口角 | 手术床+Ia+Ib+VIII | 多向 |

### 2.2 舌癌

| 舌位 | CTV 关键点 | 颈部 |
|------|----------|------|
| 舌尖 | 舌尖残端+1.5cm | **双侧 Ia+Ib+II** |
| 舌背 | 舌背术腔+1.5-2cm，深部≥1cm | 同侧 Ib+II+III（偏侧）或双侧 |
| 舌腹 | 舌腹术腔+口底接触面+1.5cm，**同侧口底必须纳入** | 同侧 Ib+II+III |
| 舌缘偏前 | 同侧舌体+口底前 | Ib+II |
| 舌缘偏后（近舌根） | 同侧舌根1-1.5cm | Ib+II+III，⚠️ 可能需 RP |

### 2.3 口底癌

| 口底 | CTV | 颈部 |
|------|-----|------|
| 前口底 | 术腔+1.5cm，对侧前口底1cm，双侧下颌骨舌侧骨膜 | **双侧 Ia+Ib+II+III** |
| 侧口底 | 术腔+1.5cm，同侧下颌骨舌侧骨膜，同侧舌腹 | 同侧 Ib+II+III |

### 2.4 牙龈癌

| 牙位 | 颈部 |
|------|------|
| 下前牙（31-43） | 双侧 Ia+Ib |
| 下后牙（44-48） | 同侧 Ib+II+III |
| 上前牙（11-23） | Ib+II（同侧为主） |
| 上后牙（24-28） | 同侧 Ib+II+III |

颊侧牙龈→CTV 向外含颊肌+皮下；舌侧→向内口底/硬腭；累及龈颊沟→全层过渡带。

### 2.5 颊黏膜癌

| 部位 | 边界 | CTV 要求 |
|------|------|---------|
| **前颊** | 口角-前磨牙 | 颊肌+皮下+面动脉旁软组织（**IX区**），可能累及唇 |
| **后颊** | 磨牙-翼下颌缝 | Buccinator 肌全层，向后→翼下颌缝和 RMT，⚠️ 可能需 RP |

> 前颊淋巴→Ib+II；后颊淋巴→II+III 并可沿翼下颌缝向深部上行。

### 2.6 硬腭癌

| | 硬腭前 1/2 | 硬腭后 1/2 |
|---|---|---|
| IX 区 | ✅ | ✅ |
| Ib 区 | ✅ 同侧 | ✅ 同侧 |
| RP | ❌ | ⚠️ **同侧 RP 外侧组** |
| VIII 区 | — | ⚠️ 累及上颌结节→考虑 |

## 三、重建皮瓣的靶区处理

> ⚠️ **全球尚无共识。** 九院当前实践：全皮瓣照射。

| 皮瓣类型 | CTV 处理 |
|---------|---------|
| 游离前臂/ALT | 皮瓣全照，组织床（皮瓣下）是 CTV |
| 腓骨瓣（下颌重建） | 腓骨不是 CTV，周围软组织含入 CTV |
| 胸大肌皮瓣 | 肌腹附着点和蒂部周围软组织纳入 |

**口底区**：口腔癌术后 PORT **同侧口底必入 CTV-60**。前口底近中线→双侧口底必照。

**下颌骨重建**：重建板周围软组织入 CTV，截骨端两侧各外放 1cm。

## 四、颈部淋巴结——术后 CTV

### 口腔癌术后颈部

| 原发部位 | cN0 | pN1（无 ENE） | pN2+ / ENE+ |
|---------|-----|-------------|------------|
| 舌（前 2/3） | 同侧 Ib+II+III | I-II-III-IV | I-II-III-IV-V + 对侧 I-II-III |
| 口底 | 同侧 Ib+II+III | I-II-III-IV | I-II-III-IV-V |
| 牙龈（下） | 同侧 Ib+II+III | I-II-III-IV | I-II-III-IV-V |
| 颊黏膜 | 同侧 Ib+II+III | I-II-III-IV | I-II-III-IV-V |
| RMT | 同侧 Ib+II+III+RP | I-II-III-IV+RP | I-II-III-IV-V+RP |

### 择区覆盖 vs 全颈

| 因素 | 全颈 | 择区 |
|------|:--:|:--:|
| 原发部位偏侧化，对侧 LN(-) | — | ✅ |
| ENE(+) / ≥3 个阳性 LN | ✅ | — |
| 口腔癌（Ib 必覆盖） | 仅患侧全颈 | |

> 完整对侧颈预防决策见 `对侧颈预防照射决策树.md`（独立 Skill）。

### 间室之间的"门"

口腔癌不会像鼻咽癌那样常规侵犯 V 区。但当 II 区阳性→CTV 跨过胸锁乳突肌后缘 5-10mm（II/V 门），不照全 V 区。

| 门 | 解剖位置 | 何时覆盖 |
|----|---------|---------|

## 附：失败模式证据

**Cho et al. (2019)**：114例零皮瓣内复发。但 GORTEC/HNCIG 共识仅 55.6% 同意豁免皮瓣。九院当前：全皮瓣照射。
